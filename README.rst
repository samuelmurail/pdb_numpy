Pdb Numpy
==============

Pdb_Numpy is a python library allowing simple operations on pdb coor files.

* Source code repository:
   https://github.com/samuelmurail/pdb_numpy


Main features:
--------------

* Basic pdb operations
   - pdb wrapper
   - selection tools
